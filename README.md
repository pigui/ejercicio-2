#REACT CON GOOGLE MAPS

##_LEVANTAR EL PROYECTO_

- npm install (para instalar dependencias)
- crear un archivo .env o renombrar el .env.example y  añadir la "key" de google
- npm run start

##_INFORMACIÓN DEL PROYECTO_
decidi crear 2 stores independientes "markerReducer" y "searchReducer".
el primero guardara la informacion general del Mapa y los markers y el segundo toda la info sobre la busqueda.
